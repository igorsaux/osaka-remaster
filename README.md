# Kasuga Ayumu no Tsuhan Seikatsu Remake

A remake of the original game **Kasuga Ayumu no Tsuhan Seikatsu**, rebuilt using the **Godot Engine**.

## 🛠 Assets & Tools

The assets for this project are extracted from the original game.

- **Original Game:** [Download from Archive.org](https://archive.org/details/osakasimulator)
- **Extraction Tool:** [KATS-tools](https://github.com/igorsaux/KATS-tools) - used to extract assets from the original game.
